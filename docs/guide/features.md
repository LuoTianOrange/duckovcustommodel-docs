# 基本功能

## 核心功能特性

### 自定义模型替换
允许玩家使用自定义模型替换游戏中的角色模型，提供完全的个性化体验。

### 模型选择界面
提供图形化界面，方便玩家浏览和选择可用模型。支持模型卡片预览、详细信息显示。

### 模型搜索
支持通过模型名称、ID 等关键词搜索模型，快速定位所需内容。

### 模型管理
- 自动扫描并加载模型包
- 支持多个模型包同时存在

### 增量更新
使用哈希缓存机制，只更新变更的模型包，大幅提高刷新效率。

### 多对象支持
每个模型目标类型（ModelTarget）可对应多个游戏对象。

### 快速切换
支持在游戏中快速切换模型，无需重启游戏。

## 支持的模型目标类型

| 类型 | 说明 |
|------|------|
| Character | 玩家主角模型 |
| Pet | 宠物模型 |
| AICharacter | AI 角色模型 |
